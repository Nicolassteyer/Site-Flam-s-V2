# Flam's

Site vitrine Next.js + Tailwind CSS pour une enseigne de restauration.

## Installation

```bash
npm install
npm run dev
```

## Mise en ligne avec GitHub + Vercel

```bash
git init
git add .
git commit -m "Initial commit - Flam's website"
git branch -M main
git remote add origin https://github.com/TON-USER/flams.git
git push -u origin main
```

Ensuite :
1. Connecte ton repo GitHub à Vercel
2. Importe le projet
3. Vercel détectera automatiquement Next.js
4. Déploie

## Images à remplacer

Place tes vrais fichiers ici :
- `public/images/hero.svg`
- `public/images/dish-1.svg`
- `public/images/dish-2.svg`
- `public/images/restaurant-1.jpg`
